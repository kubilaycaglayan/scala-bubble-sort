## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Scala
- sbt
### Setup
- `git clone git@github.com:kubilaycaglayan/scala-bubble-sort.git`
- `git checkout bubble-sort`
### Usage
- `sbt run`
### Run tests
- `sbt test`
